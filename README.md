# 🤖 Nayana's AI Chatbot

My personalized AI chatbot built with Python and Streamlit! This is my first **MAJOR Python project** with 15+ features.

## ✨ Features

### 🎮 Games & Fun
- 🎲 **Number Guessing Game** - Guess the number between 1-10
- 📝 **Python Quiz** - Test your Python knowledge
- 🤔 **Riddles** - Solve fun riddles
- 📖 **Stories** - Hear interesting stories
- 🔮 **Fortune Teller** - Get your fortune

### 💬 Chat Features
- 👋 **Personalized Greetings** - Knows your name (Nayana!)
- 🐍 **Python Discussions** - Talks about your favorite language
- 😂 **Jokes** - 10+ programming jokes
- 💡 **Facts** - Interesting tech facts
- 💪 **Motivation** - Encouraging messages
- 🌟 **Compliments** - Positive reinforcement

### 🎨 Customization
- ⚫ **Black Theme** - Your favorite color!
- 💬 **Chat History** - Remembers conversation
- 📊 **Message Counter** - Tracks your activity
- 🗑️ **Clear Chat** - Reset conversation anytime
